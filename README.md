# 📱 Diaytar

Ce projet est une application mobile développée avec **React Native**. Il a été initialisé via la `@react-native-community/cli`.

## 🚀 Fonctionnalités
- Intégration de composants natifs.
- Navigation fluide entre les écrans.
- Gestion d'API et Authentification.

## 🛠️ Stack Technique
- **Framework** : React Native
- **Langage** : JavaScript / TypeScript
- **Style** : StyleSheet / [Styled Components si utilisé]
- **Icônes** : React Native Vector Icons

## 📦 Installation et Lancement

Assurez-vous d'avoir configuré votre environnement de développement (Node.js, Android Studio/Xcode).

1. **Cloner le projet** :
   ```bash
   git clone https://github.com/Moudieng209/Mobile-Diaytar.git


# Getting Started

> **Note**: Make sure you have completed the [Set Up Your Environment](https://reactnative.dev/docs/set-up-your-environment) guide before proceeding.

Etape1: Installer les dépendances
npm install
# ou
yarn install

Etape2: Lancer l'application
npx react-native run-android
# ou
